# Simple Loan Management System

A simple loan management system that allows users to submit loan requests, process approval workflows.

## Tech Stack

* **Backend**: Spring MVC (Java)
* **Frontend**: Flutter
* **Database**: MySQL

## Features

* Submit loan applications
* Loan approval workflow
* Payment confirmation
* Loan status

## Project Structure
```
.
├── backend/
├── frontend/
└── Database/
    └── schema.sql
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/simple-loan-management-system.git
cd simple-loan-management-system
```

### 2. Configure database connection

### 3. Import the database schema

Import the schema located at:

```
Database/schema.sql
```

into your MySQL database.

### 4. Run the backend

### 5. Run the frontend

Run the Flutter application:

```bash
flutter pub get
flutter run
```

## Notes

This project was developed as a learning exercise while exploring Spring MVC and Flutter integration.
